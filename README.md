# ABTI - Academic Behavior Type Indicator

**测测你的科研人格**

👉 **[立即体验：abti.fun](https://abti.fun)**

一个面向科研工作者的趣味人格测试，覆盖从硕博研究生到博后、青年教师、PI 等各阶段。通过 30 道科研场景题，从 5 大模型、15 个维度分析你的科研行为模式，匹配 26 种科研人格类型。

> 本测试仅供娱乐，别拿它当学术评估、毕业依据、职称材料或基金申请的参考。

---

## 五大模型

| 模型 | 维度 |
|------|------|
| 科研驱力 | 好奇心纯度 · 学术野心 · 选题冒险度 |
| 工作模式 | 规划性 · 完美主义 · 工作节奏 |
| 学术社交 | 协作偏好 · 学术社交 · 权力距离 |
| 心态韧性 | 抗拒稿能力 · 自我认同 · 焦虑管理 |
| 知识产出 | 输入输出比 · 写作体质 · 创新倾向 |

## 技术说明

- 纯前端单文件应用，无需后端
- 每维度 2 题，共 30 题 + 2 道隐藏门控题
- 每题 3 个选项对应 1/2/3 分，每维度总分 2-6 映射为 L/M/H
- 用户 15 维向量与 24 种预设模式做曼哈顿距离匹配
- 相似度 < 60% 触发兜底类型 404
- 隐藏的"退坑门控"触发 QUIT 类型
